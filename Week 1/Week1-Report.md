## Progress Breakdown

### Lab 1 — Welcome to Python
Completed all core exercises: variable declaration, basic function writing, FizzBuzz, and Fahrenheit-to-Celsius converter. Also completed all bonus challenges including the regular and SUPER tic-tac-toe boards and the square root formatter.

### Lab 2 — Reviewing Syntax
Worked on identifying and correcting syntax errors in conditional blocks and loops. Implemented the 99 Bottles of Beer song and a binary search-style mystery number finder. Also tackled the Collatz sequence problem.

### Lab 3 — Strings
Explored string operators, formatting (`%`, `.format()`, f-strings), and `split`/`join`. Worked through several word-puzzle challenges (cyclone words, triangle words, triad words, surpassing words) using a system dictionary file.

### Lab 4 — Lists, Tuples, and Dicts
Practiced slicing, tuple packing/unpacking, dictionary operations, and list comprehensions. Implemented GCD using tuples, dict flipping and key merging, Pascal's triangle generation and pretty-printing.

### Lab 5 — Functions
Studied all argument types: positional, keyword, default, variadic (`*args`, `**kwargs`), and keyword-only. Explored scope, mutable default arguments, docstrings, and type annotations. Implemented `speak_excitedly`, `average`, and `autodoc`.

### Lab 6 — OOP Basics
Built classes from scratch: `Monkey`, `Hero`, `BadGuy`, `Ape`, `Gorilla`, and `Nut`. Implemented constructors, instance methods, inheritance with `super()`, magic methods (`__str__`, `__eq__`, `__ge__`), and static/class methods. Used `functools.total_ordering` for complete comparison ordering.
